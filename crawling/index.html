Googlebot requests a page URL (use BeautifulSoup to show an example) then produces protocol buffer (similar to JSON). This protocol buffer will then be used for indexing.


Where does Googlebot get a page URL to request?

  • Internal Links
  • External Links
  • XML Sitemaps
  • RSS / Atom Feeds
  • Unlinked URLs
  • ???

What stops a URL from reaching the indexing step?

Robots.txt Disallow Crawls

HTTP Status Codes != 200

  • Soft-404
  • 301 & 302
  • 4xx
  • 5xx & 429
